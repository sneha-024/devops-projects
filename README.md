# Project 2 — Dockerized App with CI/CD Pipeline

## 🎯 Motive
Automate everything — code push karo, baki sab automatic!
No manual deployment. No human error.

## 🏗️ Architecture
User → GitHub Push → GitHub Actions → Docker Build → DockerHub → AWS EC2

## 🛠️ Tech Stack
- Docker & Dockerfile
- Docker Compose
- GitHub Actions (CI/CD)
- DockerHub Registry
- AWS EC2
- NGINX

## 🔄 Pipeline Flow
1. Code push to GitHub
2. GitHub Actions trigger automatically
3. Docker image build
4. Push to DockerHub
5. Deploy to AWS EC2

## ✅ What I Built
- Containerized web app using Docker
- Automated CI/CD pipeline with GitHub Actions
- DockerHub integration for image registry
- NGINX as web server inside container

## 💡 What I Learned
- Docker containerization
- Writing Dockerfile and docker-compose.yml
- GitHub Actions workflow
- GitHub Secrets for secure credentials
- Automated deployments
